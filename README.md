# 🏧 ATM Machine Simulation (Java)

A console-based ATM Machine Simulation built using **Core Java**, replicating real-world banking operations like authentication and transactions.

---

## 📌 Overview
This project simulates an ATM system where users can securely log in and perform banking operations such as balance checking, withdrawing, and depositing money.

---

## ✨ Features
- 🔐 PIN Authentication  
- 💰 Balance Inquiry  
- 💸 Cash Withdrawal (₹100 multiples only)  
- 💵 Deposit Functionality  
- ⚠️ Error Handling (invalid PIN, insufficient balance, etc.)  
- 📋 Menu-driven interface  

---

## 🛠️ Tech Stack
- Java (Core)
- Control Statements (if-else, switch)
- Loops

---

## ▶️ How to Run
```bash
javac ATM.java
java ATM
